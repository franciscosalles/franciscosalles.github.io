Relação de Códigos Usados < HTML > Este tag inicia a função para executar uma página em Html.

< HEAD > Este tag inicia a função de cabeçalho da página.

< TITLE > Este tag executa o título da página que fica no cabeçalho.

< BODY > Este tag inicia a função de Corpo da Página, dentro dele vai todo o conteúdo dela.

< BODY BG COLOR="BLACK" > Este tag especifica uma cor de fundo (bg=backgrownd) para o Corpo da Página.

< BODY BACKGROWND="..." > Este tag especifica uma imagem de fundo para o Corpo da Página.

< FONT > Este tag é usado para formatação de texto na Página.

< FONT FACE=".." > Este tag aponta um estilo de letra para o texto.

< FONT SIZE=".." > Este tag aponta um tamanho de letra para o texto.

< FONT COLOR="." > Este tag aponta uma cor de letra para o texto.

< B > Este tag define a letra do seu texto em Negrito.

< I > Este tag define a letra do seu texto em Itálico.

< CENTER > Este tag define que seu texto será alinhado no centro da Página.

< RIGHT > Este tag define que seu texto será alinhado à direita da Página.

< LEFT > Este tag define que seu texto será alinhado à esquerda da Página.

< IMG SRC="..." > Este tag serve para inserir uma Imagem na sua Página.

< P > Este tag define o fim de um parágrafo e o início do próximo.

< P ALIGN="..." > Este tag define um alinhamento específico ao parágrafo.

< MARQUEE > Este tag é usado para executar um efeito deslizante no texto.

< MARQUEE BEHAVIOR > Este tag especifica que o efeito deslizante será de ida e volta na Página.

< MARQUEE SCROLLDELAY=".." > Este tag especifica uma velocidade para o efeito deslizante.

< BR > Este tag abre o espaço de uma linha para o texto que estiver abaixo.

< HR > Este tag insere uma linha divisória no texto da Página.

< HR SIZE="..." > Este tag especifica um tamanho para a linha divisória.

< HR COLOR="..." > Este tag especifica uma cor para a linha divisória.

< BLINK > Este tag faz o texto ou a imagem piscar.

< A HREF="..." > Este tag insere um Link em um determinado texto da Página.
