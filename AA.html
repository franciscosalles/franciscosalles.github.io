INSTRUÇÕES PARA CRIAR UMA PÁGINA HTML


PRIMEIROS PASSOS

LEMBRE-SE SEMPRE DE COLOCAR TODOS OS TAGS
PRINCIPAIS, ELES SÃO A BASE DE TODA PÁGINA NA INTERNET E
UM ERRO PODE COMPROMETER TODO O SEU TRABALHO. OS TAGS
PRINCIPAIS SÃO "HTML", "HEAD", "TITLE" E "BODY".

ATENÇÃO AO FECHAR OS TAGS

NUNCA SE ESQUEÇA DE FECHAR OS TAGS USANDO A "/" (BARRA)
PARA ENCERRAR A FUNÇÃO, SE NÃO FIZER ISTO SUA PÁGINA
ACUSARÁ ERRO. OS TAGS "BR" (PARA DAR O ESPAÇO DE UMA
LINHA) E "HR" (PARA INSERIR UMA BARRA) SÃO EXCESSÕES!

FORMATANDO O CONTEÚDO

OS TAGS DE FORMATAÇÃO SÃO MUITO IMPORTANTES, SÃO ELES
QUE VÃO MOSTRAR O DESIGN DO SEU WEB SITE. USANDO O TAG
"FONT" FIQUE ATENTO AS ESPECIFICAÇÕES SIZE, FACE E COLOR.
UMA DICA: USE A ESPECIFICAÇÃO "SIZE" APENAS EM TEXTOS
PRINCIPAIS OU DE DESTAQUE, NOS DEMAIS PREFIRA O PADRÃO.

INSERINDO IMAGENS

PARA INSERIR IMAGENS NA SUA PÁGINA USE O TAG "IMG", DÊ UM
ESPAÇO E USE A ESPECIFICAÇÃO "SRC" (ABREVIAÇÃO DE SEARCH)
QUE SIGNIFICA PROCURAR, ESTA ESPECIFICAÇÃO VAI PROCURAR
O ENDEREÇO (OU URL) DE ONDE ESTIVER LOCALIZADA A IMAGEM
QUE QUEREMOS INSERIR. VEJA O EXEMPLO AO FINAL DA PÁGINA.

CRIANDO LINKS

VOCÊ PODE CRIAR LINKS COM OUTRAS PÁGINAS DA INTERNET,
PARA ISTO DEVE USAR O TAG "A REF=".."" E DENTRO DAS
ASPAS DEFINIR O ENDEREÇO DO SITE DESTINO DO NOSSO LINK.
